Project Highlights: 

- Built with React using modern functional components

- Uses useState to manage game state (player choice, computer choice, score, and result)

- Uses props to pass data and actions between components

- Game logic separated from UI for better organization and readability

- Implements random computer choices for each round

- Displays round results and updates the scoreboard dynamically

- Allows the player to reset the game and score at any time

- Clean and responsive UI design

- Includes visual feedback for player and computer moves
